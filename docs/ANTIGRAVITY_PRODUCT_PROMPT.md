# Antigravity Product Add Prompt

**Task**: Add demo products to WooCommerce marketplace at https://beardsandbucks.com/wp-admin

---

## Instructions

You will receive product data in this format:

```
Product Name: [name]
Description: [full description]
Short Description: [1-2 sentence summary]
Price: [USD amount]
Condition: [Excellent/Good/Needs Tune]
Bow Type: [Compound/Recurve/Crossbow/N/A]
Brand: [brand name]
Year: [YYYY]
Tags: [tag1, tag2, tag3]
Image: [image filename or URL]
```

---

## Steps to Complete

1. Go to https://beardsandbucks.com/wp-admin
2. Navigate to Products → Add New
3. Fill in each field with the provided data:
   - **Product name** = Product Name
   - **Description** = Description (paste into editor)
   - **Short description** = Short Description
   - **Regular price** = Price
   - **Stock quantity** = 1
   - **Condition** = Condition (dropdown)
   - **Bow Type** = Bow Type (dropdown)
   - **Brand** = Brand (text field)
   - **Year** = Year (number field)
   - **Product tags** = Tags (text field, comma-separated)
   - **Product category** = Demo Content (checkbox only)
4. Upload image as featured image
5. Click Publish
6. Repeat for next product

---

## Notes

- Only add to "Demo Content" category — do not select any other categories
- Stock quantity is always 1 (single used item)
- Leave sale price blank
- Do not add or modify any content — use exactly what's provided
- Confirm each product publishes successfully before moving to next
